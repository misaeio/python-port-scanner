# python-port-scanner
Python Port Scanner

A basic Python tool to scan open ports on a system and explore network security fundamentals. Designed for learning and safe practice on your own machine.

Features

Scans a range of ports (default: 1–1024) on a target IP

Identifies which ports are open

Uses Python socket programming — no external modules required

Safe to run on your own machine

Getting Started
Prerequisites:

Python 3.x installed on your system

How to Run:

Clone this repository:

git clone https://github.com/misaeio/python-port-scanner.git

Navigate to the project folder:

cd python-port-scanner

Run the scanner:

python scanner.py

Watch the open ports print to the terminal

Only scan devices you own. Scanning external networks without permission is illegal.

Example Output
Scanning 127.0.0.1...
Port 22 is open
Port 80 is open
Port 443 is open
Scan complete!
